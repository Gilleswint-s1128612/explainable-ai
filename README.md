# Explainable AI: How do SHAP and DiCE differ in their ability to detect possible bias in a loan-approval model?

This repository contains the implementation and experimental setup for a comparative study between **SHAP** (SHapley Additive exPlanations) and **DiCE** (Diverse Counterfactual Explanations) in the context of personalized banking.

## Project Overview
This project uses the Loan Approval Classification Dataset https://www.kaggle.com/datasets/taweilo/loan-approval-classification-data,o evaluate how different XAI methods detect algorithmic bias in a loan-approval model 

## Running project
This project was made in google colab, to run you need to first you must download the loan.csv file from this repository and upload into your google drive
Then allow your google colab to mount drive and press run all. Make sure all imports are downloaded
